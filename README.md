## Available Scripts

In the project directory, you can run:
<br/>
To start the server
### `cd server`
### `npm install`
### `node index.js`

To start the client
### `cd curd`
### `npm install`
### `npm start`


<br/>
Then navigate to your browser http://localhost:3000/ to view the app


<br/>
This application is hosted at https://employee-management-mysql.netlify.app/
